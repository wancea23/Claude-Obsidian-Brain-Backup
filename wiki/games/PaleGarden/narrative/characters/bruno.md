# Bruno — Character Arc

A dog. Appeared Day 4. Origin unknown.

## Behavior
Follows Elias if named. Refuses the north corner of the field.
Barks once per day in the direction of the back field.
Trusts Elias completely. This is the problem.

## Key flag
`bruno_named` — set when player chooses to keep and name him.
If false: Bruno wanders off by Day 7.

## Act 3 significance
If named, the plant knows his name before Elias says it.
How it knows is never explained.
Bruno's collar persists as an item if he is sacrificed. It moves once when picked up.
